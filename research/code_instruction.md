# Kod Instruction Dataset'leri

## Mevcut Dataset'ler
| Dataset | Boyut | Açıklama |
|---------|-------|----------|
| CodeAlpaca | 20K | Kod yazma instruction |
| Evol-Instruct-Code | 110K | Kod evrim instruction |
| CodeLlama dataset | 500K | CodeLlama eğitim |
| Magicoder | 75K | OSS-Instruct |
| CodeFeedback | 140K | Kod review |

## Kod Görev Kategorileri
1. **Kod yazma**: Prompt → kod
2. **Kod tamamlama**: FIM (fill-in-the-middle)
3. **Hata bulma**: Bug detection
4. **Kod açıklama**: Code → açıklama
5. **Refactoring**: Kod iyileştirme
6. **Test yazma**: Unit test generation
7. **Dökümantasyon**: Docstring generation
