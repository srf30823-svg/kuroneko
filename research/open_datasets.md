# Açık Dataset'ler — RedPajama, Dolma, FineWeb

## RedPajama
- **Boyut**: 100B+ token
- **Lisans**: Apache 2.0
- **İçerik**: Common Crawl, Wikipedia, GitHub, ArXiv, StackExchange
- **Avantaj**: LLaMA eğitim verisi, iyi belgelenmiş
- **Dezavantaj**: İngilizce merkezli

## Dolma
- **Boyut**: 3T token
- **Lisans**: ODC-BY
- **İçerik**: Common Crawl, Wikipedia, GitHub, ArXiv, StackExchange, C4
- **Avantaj**: En büyük açık dataset, iyi dokümante
- **Dezavantaj**: İngilizce merkezli

## FineWeb
- **Boyut**: 15T token
- **Lisans**: ODC-BY
- **İçerik**: Common Crawl (filtreli)
- **Avantaj**: Yüksek kalite filtreleme, dedup
- **Dezavantaj**: İngilizce merkezli

## FineWeb-2
- **Boyut**: Çok dilli
- **Lisans**: ODC-BY
- **İçerik**: Common Crawl (çok dilli, filtreli)
- **Avantaj**: Türkçe dahil 100+ dil
- **KuroNeko için**: Ana Türkçe kaynak
