# Veri Çeşitliliği
> Adım 26 | Tarih: 2026-05-21

## Neden Önemli?
- Tek tip veri = Dar model
- Çeşitli veri = Genel yetenekli model
- Overfitting önler
- Genelleme artar

## Çeşitlilik Boyutları
1. **Konu çeşitliliği:** Kod, sohbet, özetleme, çeviri
2. **Dil çeşitliliği:** Türkçe + İngilizce
3. **Zorluk çeşitliliği:** Kolay, orta, zor
4. **Format çeşitliliği:** Instruction, chat, QA, code
5. **Kaynak çeşitliliği:** GitHub, Wikipedia, özel üretim
